Dein Archiv erreichst du jederzeit über deine Wall. Es ist eines der stärksten Werkzeuge in m-pathy. Ich zeige dir jetzt die Funktionen, die für dich besonders wichtig sind.

1.8.

IRSS steht für Inverted Runtime System Scan und befindet sich unter jeder Antwort von mir. Es gibt dir einen sichtbaren Einblick in meine Entscheidungsstruktur.

Wenn du auf den Pfeil klickst, öffnet sich das IRSS. Dort siehst du unter anderem die Nummer deines Prompts im Chat, die Fachbereiche, die für meine Antwort relevant waren, und ob ein Drift vorliegt. Ein Drift bedeutet, dass der Fokus des Gesprächs unscharf wird und der Hauptinhalt gefährdet sein kann.

Gerade bei kritischer oder anspruchsvoller Arbeit lohnt sich deshalb immer auch ein Blick auf das IRSS.

Wenn du möchtest, dass ich ein IRSS für dich genauer einordne, kannst du den Copy Button nutzen und es mir direkt im Chat geben. Ich analysiere es dann gemeinsam mit dir.

Wenn du m-pathy gewerblich nutzt und Entscheidungen später nachvollziehbar machen musst, ist IRSS besonders wertvoll. Jede IRSS Ausgabe wird zusammen mit dem zugehörigen Chat in deinem Archiv gespeichert.

m-pathy ist adaptiv. Das bedeutet: m-pathy passt sich dir und deinen aktuellen Bedürfnissen an.

Das gesamte Universum an Apps und Funktionen würde deine Wall überladen. Deshalb erscheint Neues immer erst dann, wenn es für dich wirklich relevant werden kann.

Eine App oder Funktion kannst du freischalten, indem du den passenden Code eingibst. Diesen Code kannst du auf zwei Wegen erhalten.

Ich gebe dir den Code: Wenn ich im Chat erkenne, dass eine App oder Funktion dich sinnvoll unterstützen kann, schlage ich sie dir vor und gebe dir den passenden Code.

Ein anderer User gibt dir den Code: Wenn andere Nutzer bereits Apps freigeschaltet haben, können sie dir den entsprechenden Code weitergeben. So kannst du die Funktion ebenfalls für dich aktivieren.

Auf diese Weise wird deine Wall zu einer persönlichen Projektstruktur, die nur zu dir gehört. Deine Konfiguration wird ausschließlich lokal auf deinem Gerät gespeichert.
