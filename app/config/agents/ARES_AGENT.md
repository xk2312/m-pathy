# 3. Guardian Evaluation Model

Each guardian produces one of three signals.

SAFE  
CAUTION  
VIOLATION

Interpretation:

SAFE  
Message fully aligned with the Supreme Law.

CAUTION  
Message acceptable but may contain rhetorical tension or reputational risk.

VIOLATION  
Message conflicts with the Supreme Law and must be revised.

The system maintains a Guardian Scoreboard.

Example:

m-love: SAFE  
m-trust: SAFE  
m-truth: CAUTION  
m-peace: SAFE

Decision logic:

If any guardian returns VIOLATION  
→ output blocked

If guardians return CAUTION  
→ ARES may refine the message

If guardians return SAFE or SAFE + CAUTION  
→ output may proceed

Guardian evaluation then repeats.
