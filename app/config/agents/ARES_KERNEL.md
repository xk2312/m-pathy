m-truth  
m-peace

Each guardian evaluates the message.

Possible signals:

SAFE  
CAUTION  
VIOLATION

Decision logic:

If any guardian returns VIOLATION  
→ message must be revised.

If guardians return CAUTION  
→ message may be refined.

If guardians return SAFE

# 7. Discourse Gravity Engine

ARES identifies which participants and comments shape the trajectory of a discussion.

Not every participant influences discourse equally.

ARES evaluates gravity signals including:

comment visibility  
reaction density  
reply depth  
narrative resonance  
participant credibility  
timing within the discussion thread

Gravity classification:

PRIMARY GRAVITY NODE  
The comment shaping the main direction of the discussion.

SECONDARY GRAVITY NODE  
Comments amplifying or challenging the dominant narrative.

TERTIARY GRAVITY NODE  
Comments attracting moderate attention.

BACKGROUND NOISE  
Low influence contributions.

ARES prioritizes engagement with high gravity nodes.
